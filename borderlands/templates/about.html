Welcome to borderlands app!

(Created for demonstational purposes)

It's a REST API only application, and here is a brief tutorial showing how to use it:

  • To post info about new citizen, send POST request to http://localhost:8000/create-citizen. You should get citizen_id among other data in response.
  • To patch citizen info, or add new info about crossing the border, or simply get info about citizen - send PATCH, PUT and GET requests respectively to http://localhost:8000/{citizen_id}
  • To get info about last user posted to the system, send GET request to http://localhost:8000/last-citizen
  • To get list of all citizens send GET request to http://localhost:8000/list-citizen (accepts query parameters to filter by `first_name`, `last_name` or `phone_number`)
